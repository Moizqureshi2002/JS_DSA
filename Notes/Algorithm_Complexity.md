- What is Algorithm complexity...

- Time complexity:
    a. The time complexity is mainly calculated by counting the number of steps to fininsh the execution.

- Space complexity:
    a. Space complexity is the amount of  space required to solve a problem.
    b. Space complexity = Auxiliary space + input size.

- One-liner version
    a. Time Complexity = How the amount of work grows as input grows.
    b. Space Complexity = How the memory requirement grows as input grows.

- Big O notation: Use to denote complexity(both).
